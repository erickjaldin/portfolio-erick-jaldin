# Detector de Traje (IA) 👔

Este projeto consiste em um modelo de **Machine Learning** (Visão Computacional) treinado para identificar e classificar vestimentas em tempo real.

### 🚀 Acesse o Modelo
Você pode testar o funcionamento do detector diretamente pelo navegador através do link abaixo:
👉 [Testar no Teachable Machine](https://teachablemachine.withgoogle.com/models/OH_yVPeqk/)

### 📊 Detalhes Técnicos
* **Ferramenta:** Google Teachable Machine
* **Tipo de IA:** Image Model (Classificação de Imagem)
* **Objetivo:** Diferenciar trajes formais (terno) de outros tipos de vestimenta para estudos de reconhecimento de padrões e automação.
